# QuickMythril

I am a primary maintainer and an administrator of
[Qortium](https://qortium.app), an independent open-source Qortal-family
blockchain and QDN platform currently running a live Previewnet.

- [Qortium Core](https://github.com/QortiumDev/qortium-core) - Java blockchain,
  QDN node, and Previewnet infrastructure
- [Qortium Home](https://github.com/QortiumDev/qortium-home) - desktop and
  Android wallet, node, and QDN application shell
- [Qortium Development](https://github.com/QortiumDev) - the full public
  ecosystem and QDN apps

I also contribute to Qortal-family and other open-source projects.
